Rusted Script Syntax

Rusted Script has a simple syntax, and is easy to learn for beginners. It is a single threaded language with a very fast runtime. It is still being developed, and if you have any suggestions dont hesitate to open a new issue, and also if you spot any bugs, you can open an issue.

            
            # Rusted Script Example
            var x = 1 + 1 # define a variable
            if x == 3 then 123 elif x == 0 then 321 else 12345 # if statement

The above would return 12345, since x is equal to 2, and the if statement checks for if it equals 0 or 3, it will run the else clause.
